# 下载对账单

> 每日零点开始生成前一天商户对账文件，文件生成需要一定时间，请滞后进行文件下载

!> 文件响应成功的HTTP状态码为`200`，如果文件正在处理中响应的HTTP状态码为`202`

**请求URL**

- `{{host}}/openApi/v1/payee/statement`


**请求方式**

- GET

**请求参数**


| 参数名      | 必选 | 类型   | 默认值 | 说明                   |
| ----------- | ---- | ------ | ------ | ------------------ |
| day   | n    | string  | T-1      | 对账单日期，格式：YYYY-mm-dd，默认为前一天 |

**返回说明**

返回文件响应，文件格式为 `xlsx`;
