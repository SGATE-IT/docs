选择【Manage Users】模块，你可以看到该商户的所有用户信息

可以点击【用户名称】查看用户信息详情

可以在【Manage Users】创建商户的新用户

![img](../_media/user-4.png ':size=45%')
![img](../_media/user-5.png ':size=40%')
