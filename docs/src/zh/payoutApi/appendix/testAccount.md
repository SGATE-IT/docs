# 测试账户

::: warning 注意
1. 测试账户只支持沙箱环境使用，生产环境所有账户都会正常进行付款。
2. 沙箱环境中测试账户不受客户自动审批影响，即使客户没有开启自动审批，依然会自动处理。
3. 沙箱环境中如果客户开启了自动审批并且使用了测试账户，审批结果同测试账户自动结果。
4. 沙箱环境中测试账户在**快捷**代付工单中**客户自动结果**都为**通过**，只有付款自动结果生效。
:::
